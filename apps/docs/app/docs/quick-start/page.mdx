# Quick start

## Installation

```sh npm2yarn copy
npm install @mesh-gradient/core
```

## Usage

Place the canvas element in body of your HTML file.

```html copy filename="index.html"
<body>
  <canvas id="canvas"></canvas>
</body>
```

Import the `MeshGradient` class and initialize the gradient by passing the canvas element.

```ts copy filename="index.ts"
import { MeshGradient } from '@mesh-gradient/core';

const gradient = new MeshGradient();

gradient.init('#canvas');
```

That's it! You can now use the mesh gradient in your project.

## Configure gradient

The mesh gradient can be configured by passing an options object as the second argument to the `init` method.

```ts copy 
import { type MeshGradientOptions } from '@mesh-gradient/core';

const options: MeshGradientOptions = {
  colors: ['#043D5D', '#032E46', '#23B684', '#0F595E'],
};

gradient.init('#canvas', options);
```
